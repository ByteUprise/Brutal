## Getting Started
1. ```Copy and paste the PaensyLib folder inside your Arduino\libraries```
1. ```git clone https://github.com/Screetsec/Brutal.git```
2. ```cd Brutal```
3. ```chmod +x Brutal.sh ```
3. ```sudo ./Brutal.sh or sudo su ./Brutal.sh ```
